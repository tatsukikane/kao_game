#  KAO GAME
 お題にあった表情をシェアする写真SNS


## 使い方
1お題の表情を確認して、写真をシェア 
2みんなの画像をいいね!で評価する 
3夜にランキングが発表される

## 詳細、動画リンク
表情を5段階評価するVision APIをPythonで実行。
いいね数とAIスコアを元にランキングを計算。
